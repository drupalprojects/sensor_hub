
A Feeds Fetcher plugin.  Will fetch data from an enabled Atrium Feed Reader in the current space of whatever node the Feed Importer is attached to that contains this Fetcher plugin. 

Originally designed for the Sensor Hub project http://drupal.org/project/sensor_hub
