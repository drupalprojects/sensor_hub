
A Feeds Fetcher plugin.  Will fetch data from the shub_sensor_board_data in the current space of whatever node the Feed Importer is attached to that contains this Fetcher plugin. 

Originally designed for the Sensor Hub project http://drupal.org/project/sensor_hub
